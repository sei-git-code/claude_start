# 8BitDo Micro 音声入力（未導入・参考情報）

出典: [Claude Code で音声入力してるやつは今すぐ 8BitDo Micro を買え](https://zenn.dev/ryu1maniwa25/articles/8bitdo-micro-voice-input-claude-code)

3,500円ほどのミニコントローラー「8BitDo Micro」をキーボードモードでMacに接続し、Dictation起動やEnter/Escなどの操作をボタン割り当てすることで、デスクから離れた場所からClaude Codeを操作できるようにする話。

## 前提
- macOS固有の手順（System Settings → Keyboard → Dictationショートカット、8BitDo Ultimate Software）
- 物理ハードウェアの購入が必要

## Windowsでの代替
`Win+H`（Windows音声入力）のショートカットをコントローラーの1ボタンに割り当てる形で近い環境を作れる。8BitDo Ultimate Softwareのキー割り当て機能はWindowsでも使える。

## なぜ未導入か
ハードウェア購入判断が必要、かつユーザーの環境はWindowsで記事の手順がそのまま使えないため見送り。導入したくなったら上記の代替案から着手する。
